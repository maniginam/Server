# Gina's HTTP Server

This is a basic HTTP server.

#####Dependencies:
* Java 8  
* Maven 4.0
* Test
    * junit-platform-launcher 1.7
    * junit-platform-jupiter 5.7
    * junit-platform-vintage 5.7

#####Run Tests
* cmd: mvn package
    
#####Execution:
* java -cp target/Server-1.0-SNAPSHOT.jar httpServer/Server
